Overview
To-Do List App is a user-friendly task management mobile application designed for seamless task organization. Users can effortlessly log in, add, and edit tasks, with each task recorded along with its timestamp for chronological tracking. Notably, the app includes a task completion feature for efficient progress monitoring. The app is non centralized data center which helps user to track their task wherever they are to increase productivity.

 

Purpose

Using and creating a to-do list application can be important for several reasons:

·      Organization: It consolidates tasks and responsibilities, simplifying the tracking of activities in a centralized location.

·      Stress Reduction: By providing a clear plan to tackle tasks, it mitigates stress and anxiety associated with feeling overwhelmed or forgetting important responsibilities.

·      Progress Tracking: Users can monitor completed tasks, fostering a sense of accomplishment and maintaining momentum.

·      Productivity Boost: The application acts as a roadmap, enhancing productivity by offering a clear list of tasks, helping users stay focused and motivated throughout the day.

·      Collaboration and Communication: In team environments, the application facilitates collaboration by enabling task sharing, responsibility assignment, and progress tracking among team members.

 

 

Implementation
The To-Do List App, built using the Flutter framework in Dart and integrated with Firebase services (Firestore and Cloud Firestore), consists of five pages designed to offer a seamless task management experience:

 

1.    Login Page:

·       Email/Username input field.

·       Password input field.

·       "Log In" button.

·       Option for password recovery or account creation.

 

2.    Sign Up Page:

·       Full Name input field.

·       Email input field.

·       Password input field.

·       "Sign Up" button.

·       Link or button for returning to the login page.

 

3.    Add Task Page:

·       Text input field for task description.

·       "Add" button to save the task to Firestore.

·       App bar with a drawer icon for navigation.

·       Drawer with user information, and options for logout, settings, contact, and about app.

 

 

 

 

 

4.    Edit Task Page:

·       Text input field for editing the task.

·       "Update Task" button to save changes to Firestore.

·       App bar with a drawer icon for navigation.

·       Drawer.

 

5.    Task List Page:

·       Display area for a list of tasks fetched from Firestore.

·       Floating action button to navigate to the "Add Task" page.

·       Edit icon for each task to navigate to the "Edit Task" page.

·       Delete icon for each task to remove it from the list and Firestore.

·       Checkbox for each task to mark it as done or incomplete.

·       App bar with a drawer icon for navigation.

 

 

6.    Drawer with user information, and options for logout, settings, contact, and about app.
